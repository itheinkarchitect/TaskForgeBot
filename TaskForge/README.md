# TaskForgeBot

Telegram-бот для управления задачами на Python с использованием aiogram 3.

## Features

- Добавление задач
- Просмотр задач
- Завершение задач
- Удаление задач
- FSM состояния

## Technologies

- Python
- aiogram 3
- Telegram Bot API

## Installation

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python bot.py
```
